### Getting Started
- Set up the [server repository](https://github.com/biantris/koa-server) (note: it needs to be running together with the frontend) 
- Clone this repo
```sh
# install dependencies
yarn
# or
yarn install

# copy .env file
cp .env.example .env

# start project
yarn dev

# see on localhost link
http://localhost:8000
```
